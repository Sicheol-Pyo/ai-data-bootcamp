3. 데이터 신뢰성 (반드시 기록)
병합 전 키 검증:
ordr의 customer_id 기준으로 cust domain merge 전 중복 검증 진행하였으며, 중복 0건으로 확인됨.
ordr_cust의 경우 product_id 기준으로 prod domain merge 전 중복 검증 진행하였으며, 중복 0건으로 확인됨.

validate="m:1" 통과 → 관계 가정(주문:상품 = m:1) 유효.

4. 제언
- category 뷰티의 매출합계가 저조한 것으로 확인되어 광고 또는 제품 구성을 변경하여 매출 증가의 변화가 필요함.
